# Supply Chain SQL Project

## Project Overview

This project is based on Supply Chain Management using SQL and MySQL Workbench.  
The main objective of this project is to analyze customer orders, supplier performance, product demand, and overall business revenue using SQL queries.

This project helps in understanding real-world business problems using Joins, Subqueries, Window Functions, Group By, Aggregate Functions, and Ranking Functions.
